The next generation mobile first framework

Born for large scale mobile/web app

Haxejs helps developers from all platforms build complex, high performance client apps for the modern web.

What is wrong with JavaScript?

  1. Lack of structure
  2. Very difficult to find dependencies
  3. All numbers are floats
  4. Monkey patching
  5. Keep on truckin' mentality
  6. Libraries are files you concatenate

... which makes it ...

  1. Difficult to optimize performance
  2. Difficult to maintain large code bases
  3. Difficult to assert correctness

Object-oriented programming

Use modern, high level, strictly typed programming language Haxe(very similar to Java) to generate elegant javascript, it provides compiler error check and code completions with many IDE tools.

Cut-edge library manager

Hundreds of haxe libraries are easy to put under your fingers, give you lots of choices.

Powerful with angularjs, and other directives, services

Angularjs is elegant and well designed, and it is the core of haxejs framework, many good javascript libraries are rewritten in angularjs way, and integrated seamlessly.

Haxejs is open source. It's hosted, developed, and maintained on GitHub.
